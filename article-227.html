M80 Framework Revisited

Around 16 years ago, back in 2007, I made something I called the M80 Framework. Just for fun I have revisited this project and have made some updates. It is still very much a proof of concept though!

Most of the updates are refactorings to bring the code up to a newer standard, but some bugs were fixed as well. The "log watcher" plugin no longer worked on newer systems, this has been fixed but in a quick and very ineffective way. No new plugins have been added at this point. The bundled README file has been converted to Markdown format.


So, here is version 0.2 for download.
