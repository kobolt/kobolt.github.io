Bad Apple!! for Epson HX-20

There exists a somewhat famous video which has a "monochrome" animation and music made by Alstroemeria Records based on the Touhou Project.

A popular activity is to make variants of this, including variants running on older computers. My contribution is for the Epson HX-20 and can be found here.

I had to use two Epson HX-20s for this due to severe hardware constraints. One HX-20 is streaming the video over the "high speed" serial link using the same method as described in my earlier post here. Some minor modifications were made in the C program to update at 30 FPS and display only 40x32 pixels, but the program running on the HX-20 is the same.

Another HX-20 is playing the audio through its piezo speaker. Here is that assembly program which plays the "Bad Apple!!" melody:

	processor hd6303

TRCSR equ $11
SRDR  equ $12
STDR  equ $13

	org $1000 ; Run from RAM
start:
	ldx #sound
loop:
	ldaa #$31 ; Advanced Sound Command
	jsr send

	ldaa 0,x ; Frequency Upper
	inx
	jsr send
	ldaa 0,x ; Frequency Lower
	inx
	jsr send

	ldaa 0,x ; Duration Upper
	inx
	jsr send
	ldaa 0,x ; Duration Lower
	inx
	jsr send

	ldaa 0,x
	cmpa #$ff
	bne loop
end:
	jmp end

send:
	dc.b $7B,$20,TRCSR ; tim #$20,TRCSR
	beq send
	staa STDR
recv:
	tst TRCSR
	bpl recv
	ldab SRDR
	rts

sound:
