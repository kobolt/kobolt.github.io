Intel SDK-85 Emulator

To help with my earlier work on the Intel SDK-85 board, I made an emulator as well. The Intel 8085 is similar to the Z80 so some code was re-used from my earlier emulators but since all opcodes are 1 byte it was eventually structured a bit different like my 6502 emulators.

The emulator can run in ether "display/keyboard" mode or "serial" mode which reflects the actual modes the board's monitor software can utilize. The "display/keyboard" mode uses curses to give a graphical view of the LED segmented display and also the keyboard buttons. In this mode input can be given by the keyboard or if curses has mouse support, the mouse can be used to click on the virtual keys. The "serial" mode runs on regular standard in/out and interfaces against the Intel 8085's built-in serial functionality at 110 baud.

As usual there is a built-in debugger with CPU tracing and breakpoint support which is accessed with Ctrl+C. The "monitor.hex" ROM is expected, but an additional ROM can also be loaded. The rest of the 64K memory space just acts as RAM. Some hacks were added in the emulator to block on user input depending on the different modes since this greatly reduces host CPU usage.
